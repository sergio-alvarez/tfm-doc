== Aplicación web de geolocalización como servicio

A continuación se muestran una serie de secciones que explicarán de manera aproximada la manera de implementar una aplicación web que ofrezca un servicio de geolocalización de tuits en base a su contenido utilizando la tecnología explorada en los prototipos y experimentos anteriores.

Es importante aclarar que, puesto que no se ha implementado la aplicación, los conceptos de diseño y análisis que se exponen en el presente proyecto podrían diferir de una aplicación real, en donde nunca se optaría por un enfoque _en cascada_ si no que se utilizaría una metodología ágil basada en iteraciones (ver <<_scrum>>), donde se irían añadiendo *user stories* con nuevas características que sirvan a su vez para refinar el diseño e implementación del sistema en cada iteración.
